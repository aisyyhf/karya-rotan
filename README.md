# Karya Rotan Indonesia

Website company profile PT Karya Rotan Indonesia untuk menampilkan profil perusahaan, kategori produk, portfolio proyek, layanan custom, katalog, dan informasi kontak.

## Teknologi

- React 19 dan TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Menjalankan proyek

```bash
npm install
npm run dev
```

## Pemeriksaan dan build

```bash
npm run lint
npm run build
```

Hasil production build tersedia di folder `dist` dan dapat dibuat ulang kapan saja melalui perintah build.

## Struktur utama

- `src/pages` — halaman website
- `src/components` — komponen layout, section, dan UI
- `src/data` — data perusahaan, produk, portfolio, dan navigasi
- `public/assets` — gambar, logo, serta katalog PDF
